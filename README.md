# Fuzzy Decision Tree рубрикатор

### Сделано Nickolay B.(waterwav3) как выпускная дипломная работа

Реализация нечёткого дерева решений (Fuzzy Decision Tree, FDT) для рубрикации
русскоязычных новостных текстов. Репозиторий содержит:

- `fdt.py` — CLI-скрипт обучения и инференса (полный pipeline: TF-IDF → χ²-отбор → FDT → pickle);
- `web_app/` — веб-приложение на FastAPI для интерактивной классификации текста
  через браузер (см. `web_app/README.md`).

## Требования

- Python 3.11+
- pip
- (опционально) Docker и docker compose — для запуска веб-приложения
x
Зависимости для обучения совпадают с зависимостями веб-приложения и
устанавливаются из `web_app/requirements.txt`:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r web_app/requirements.txt
```

## Исходный датасет

Для обучения используется открытый корпус новостей **Lenta.ru** в формате CSV.
Скачать его можно со страницы датасета на Kaggle:

<https://www.kaggle.com/code/kerneler/starter-news-dataset-from-lenta-ru-83550e4c-b>

После скачивания распакуйте архив и положите файл `lenta-ru-news.csv` в корень
репозитория (или укажите путь явно через флаг `--data-path`).

Структура CSV (минимально необходимые столбцы):

| столбец | описание |
| --- | --- |
| `text`  | текст новости |
| `topic` | рубрика (метка класса) |

Скрипт `fdt.py` автоматически отбирает 5 рубрик по умолчанию:
`Спорт`, `Экономика`, `Культура`, `Наука и техника`, `Путешествия`.

## Обучение модели

Базовый запуск с настройками по умолчанию:

```bash
python3 fdt.py --mode train
```

Скрипт последовательно выполнит:

1. чтение CSV (`--nrows` строк максимум);
2. балансировку выборки по классам (`--samples-per-class` документов на класс);
3. очистку и лемматизацию текстов (`pymorphy3`);
4. стратифицированное разбиение train/test;
5. построение TF-IDF + χ²-отбор k признаков;
6. обучение `FuzzyDecisionTree`;
7. оценку на тесте (accuracy, macro-F1, classification report, confusion matrix)
   и сериализацию pipeline в pickle.

### Основные флаги CLI

| флаг | значение по умолчанию | описание |
| --- | --- | --- |
| `--mode`               | `train`                                       | `train` — обучение, `predict` — инференс одного текста |
| `--data-path`          | `lenta-ru-news.csv`                           | путь к CSV-датасету |
| `--model-path`         | `fuzzy_text_model_5cls_tuned_12_05_0040.pkl`  | куда сохранить / откуда загрузить модель |
| `--nrows`              | `250000`                                      | максимум строк, читаемых из CSV |
| `--samples-per-class`  | `1800`                                        | балансировка корпуса (документов на класс) |
| `--topics`             | 5 рубрик по умолчанию                         | список рубрик через запятую |
| `--max-features`       | `30000`                                       | верхняя граница словаря TF-IDF |
| `--min-df`             | `2`                                           | минимальная частота термина (отсечение гапаксов) |
| `--k-best`             | `4000`                                        | размер χ²-выборки признаков |
| `--max-depth`          | `22`                                          | максимальная глубина дерева |
| `--min-gain`           | `0.0015`                                      | порог информационного выигрыша |
| `--test-size`          | `0.1`                                         | доля тестовой выборки |
| `--random-state`       | `42`                                          | сид для воспроизводимости |

Пример запуска с кастомными параметрами:

```bash
python3 fdt.py --mode train \
  --data-path /path/to/lenta-ru-news.csv \
  --model-path my_model.pkl \
  --samples-per-class 2000 \
  --max-depth 25 \
  --min-gain 0.001
```

## Инференс одного текста

После обучения модель можно вызвать в режиме `predict`:

```bash
python3 fdt.py --mode predict \
  --model-path fuzzy_text_model_5cls_tuned_12_05_0040.pkl \
  --text "Российская сборная по футболу одержала победу в товарищеском матче"
```

Скрипт выведет предсказанный класс. Если файл модели отсутствует, появится
сообщение с предложением сначала запустить обучение.

## Веб-приложение

Графический интерфейс для классификации текста в браузере находится в папке
`web_app/`. После обучения модели скопируйте полученный pickle в каталог
веб-приложения:

```bash
cp fuzzy_text_model_5cls_tuned_12_05_0040.pkl web_app/app/fuzzy_text_model.pkl
```

Запуск через Docker Compose:

```bash
cd web_app
docker compose up --build
```

После запуска интерфейс будет доступен по адресу <http://localhost:8000>.
Подробности — в `web_app/README.md`.
